# A mask detector using YOLOV5

The original repository: https://github.com/ultralytics/yolov5

**Conda enviroment**
```
conda activate <env>
conda install pip
pip freeze > requirements_ingfisica.txt
```
**PIP enviroment**
```
python3 -m venv env
source env/bin/activate
pip install -r requirements_ingfisica.txt
```


**Train**
```
train.py 
```
